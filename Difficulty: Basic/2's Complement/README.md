<h2><a href="https://www.geeksforgeeks.org/problems/2s-complement3124/1">2's Complement</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a binary string <strong>s</strong>, find its 2's complement and return the result as a binary string of the same length. </span><span style="font-size: 18px;">The 2's complement of a binary number is obtained by first flipping all its bits (0 to 1 and 1 to 0) and then adding 1 to the resulting binary number.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> s = 00000101
<strong>Output:</strong> 11111011
<strong data-start="744" data-end="760">Explanation</strong><strong>: </strong>Flip all the bits to get 11111010, then add 1 to obtain 11111011.<br></span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> s = 101
<strong>Output:</strong> 011
<strong data-start="744" data-end="760">Explanation</strong><strong>: </strong>Flip all the bits to get 010, then add 1 to obtain 011.<br></span></pre>
